## Design Specifications

| Element | Value |
|---|---|
| Background | `#F5F1E8` (cream) |
| Primary text | `#1A1A1A` (ink) |
| Accent | `#8B6F47` (bronze) |
| Emphasis | `#2D4A3E` (forest green) |
| Heading font | Cormorant Garamond, weight 500-600 |
| Body font | Inter, weight 400-500 |
| Margins | Generous. White space is a luxury signal. |
| Logo | Bottom-right corner. Small. "AISociety" |
